What is Job Shadow Portal

Job Shadow Portal is a web site developed for associates of the Company to Request for Formal Trainings/Job Shadows. This will help associates learn about other areas and Roles n Responsibilities in ISD.

How does this application Work

Open the Home Page and Select the Area you want to Request the Job Shadow in. The Area can be selected from the Images in Carousel. Once the Area is selected, click on "Create Job Shadow Request" Button. A new page to create and submit the Request gets opened. Select the Start Date and End Date for the Request. You can also provide any additional Comments. Once the request is submitted, the system will send an e-mail to the Manager of the Requested Area/Team.
